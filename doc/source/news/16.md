# News - 16 series

(release-16-0-0)=

## Release 16.0.0 - 2026-02-09

This is our annual major release! This release doesn't have any
backward incompatible changes! So you can upgrade Groonga without
migrating your existing databases. You can still use your existing
databases as-is.

### Fixes

#### Fixed a bug that causes overflow in the key of `TABLE_DAT_KEY` tables.

If you execute offline index construction against a column that has a large size value (When the size of a token is 4096 bytes after tokenization), the index may be broken.
